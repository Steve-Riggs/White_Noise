# Bundled audio files

Put your personal white noise audio files in this folder before pushing the repo to GitHub.

Supported file types:

- `.mp3`
- `.wav`
- `.ogg`
- `.m4a`
- `.flac`
- `.aac`

When the integration starts, files in this folder are copied into the configured Home Assistant media folder, by default:

```text
/media/white_noise
```

Existing files are not overwritten.
